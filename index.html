HTML Elements Reference Table

Common HTML Elements


Element Tag Element Name Description
<body> body Respresents the content of the HTML document
<h1> heading Heading elements 1-6 are used as headings and subheadings
<p> paragraph Contains and displays blocks of text
<span> span An inline container for text, can be used for styling
<ul> unordered list Creates a list in no particular order
<ol> ordered list Creates a list in a particular order
<li> list element Creates list items inside ordered and unordered lists
<div> division A container that is ued to divide an HTML document into sections
<img> image Embeds images in documents
<video> video Embeds a media player for video playback
